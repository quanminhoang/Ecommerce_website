Các bước chạy trang web
B1: Cài đặt XAMPP
B2: Clone project về folder htdocs trong Application Folder XAMPP
B3: Vào XAMPP Manage Server Start all
B4: Truy cập http://localhost/phpmyadmin/index.php?route=/server/sql
B5: Import file Ecommerce_website.sql trong folder SQL
B6: Truy cập 
    http://localhost/Ecommerce_website/Admin/ (Trang Admin)
    http://localhost/Ecommerce_website/Customer/ (Trang Customer)
